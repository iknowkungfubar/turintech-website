# TurinTech Solutions - Official Website

This repository contains the source code for the official TurinTech Solutions website. It is a single-page, static HTML website designed to showcase our full-stack AI and technical operations consulting services.

## 🌐 Live Site
The website is hosted via GitHub Pages and is live at: **[turintechsolutions.com](https://www.turintechsolutions.com)**

## 🛠️ Technology Stack
* **HTML5:** Semantic structure and content.
* **CSS3:** Custom styling, dark-mode aesthetic, and responsive flexbox/grid layouts.
* **Zero Dependencies:** No external frameworks, build tools, or databases are required. All assets (like logos and SVGs) are fully self-contained.

## 💻 Local Development
Because this is a completely static website, you do not need a local server or build environment to make edits or view changes. 

## 🔧 Recent Updates
* **SEO Enhancements:** Added meta description, Open Graph, and Twitter Card tags for better social sharing.
* **Accessibility:** Added skip-to-content link and landmark ARIA roles (banner, main) for improved keyboard navigation.
* **Bug Fixes:** Fixed logo link from placeholder `#` to proper home URL `/`.
